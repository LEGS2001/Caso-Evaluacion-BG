¿Cómo registro el servicio Palco Prepago?

Puedes registrar el servicio Palco Prepago desde la App o Banca Virtual: 

Desde la App

Ingresa a "Pagar servicios" y presiona el ícono “+” para agregar un nuevo servicio.

Busca la categoría Internet y TV pagada.

Selecciona la empresa "Palco Prepago".

Ingresa el código del servicio.

Recibirás un código de seguridad por SMS. Ingrésalo para confirmar y guardar el servicio.

El pago se procesará y el servicio quedará guardado.

Podrás ver tu comprobante, compartirlo o hacer un nuevo pago.

Desde Banca Virtual 

En el menú, selecciona Mi perfil > Matriculación de cuentas, tarjetas y servicios.

Elige Pago de servicios > Agregar nuevo servicio.

Selecciona la categoría Televisión pagada e internet y luego Palco Prepago.

Ingresa tu cédula y un nombre de referencia.

Recibirás un código de seguridad en tu correo electrónico.

Si no lo recibes, asegúrate de actualizar tus datos de contacto.

Ingresa el código para completar el registro.

¿Dónde usar el código PIN?

Una vez confirmado el pago, revisa tu comprobante. Allí encontrarás el código PIN que necesitas para acceder al contenido contratado.

Para ver el Campeonato Nacional:

Desde la App GOLTV PLAY (disponible en Google Play o App Store)

Desde tu navegador ingresando a: https://play.goltv.tv/

Selecciona el plan que pagaste y luego Plan Prepago, donde deberás ingresar el PIN de tu comprobante.

Para ver las Eliminatorias de Fútbol:

Ingresa a: www.elcanaldelfutbol.com

Haz clic en Contratar y luego baja hasta “Validar tu PIN aquí”.

Ingresa el PIN que aparece en tu comprobante.
